build
1. open build page
    - click model node and select model (Ex. Xception)
    - click Xception node

2. specify build option
    2.1) classes: 10
    2.2) File path: checkpoint/Xception/model.h5

3. build
    3.1) click run toolbar

4. check
    4.1) checkpoint/Xceptoin/model.h5


train
1. open train page
    1.1) click train toolbar

2. specify model setting
    2.1) select model (Ex. Xception)
    2.2) load_file: checkpoint/Xceptoin/model.h5
    2.3) use_checkpoint_callback: True
        2.3.1) ckpt_file_path: checkpoint/Xception/trained_model.h5
        2.3.2) monitor: loss

3. specify dataset setting
    3.1) select dataset format (Ex. directory)
    3.2) select train/test directory (Ex. dataset/cifar10/train/)
    3.3) rescale: 0.004
    3.4) target_size: 299 299

4. train
    4.1) click run toolbar

5. check 
    5.1) checkpoint/Xception/trained_model.h5


test
1. open test page
    1.1) click test toolbar

2. specify test setting
    2.1) select model (Ex. checkpoint/Xception/trained_model.h5)
    2.2) use_csvlogger_callback: True
        2.2.1) filename: checkpoint/Xception/trained_model_test.log

3. specify dataset setting
    3.1) select dataset format (Ex. directory)
    3.2) select train/test directory (Ex. dataset/cifar10/test/)
    3.3) rescale: 0.004
    3.4) target_size: 299 299

4. test
    4.1) click run toolbar

5. check
    5.1) checkpoint/Xception/trained_model.log
